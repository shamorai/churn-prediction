# 📊 Customer Churn Prediction System

🚀 A Machine Learning-based web application that predicts whether a customer will churn or not, with an interactive dashboard and feature impact visualization.

---

## 🔥 Features

- 📈 Real-time Customer Churn Prediction
- 🎯 Probability Score (Confidence %)
- 🧠 Feature Impact Analysis (SHAP-based)
- 🎨 Interactive Streamlit Dashboard
- ⚡ Fast & Lightweight Model

---

## 🛠️ Tech Stack

- Python
- Scikit-learn
- Streamlit
- NumPy
- Matplotlib
- SHAP
